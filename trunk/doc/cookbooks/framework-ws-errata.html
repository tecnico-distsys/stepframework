Errata Framework Web Services cookbook

Esta errata descreve as correcções efectuadas ao documento "Livro de Receitas Framework de aplicações com Web Services".

Correcções à versão 2008-04-03:

  • Na página 13, secção "Distribuir uma aplicação web usando Web Services", "Disponibilizar serviços remotos de invocação remota (componente ws client)", "Criar um teste de invocação do Web Service":
    • Onde se lê: "1. (...) cria e invoca serviços"
    • Deve ler-se: "1. (...) cria e invoca stubs, fazendo: importar os tipos dos stubs e da fábrica; pedir um stub à fábrica; preencher os argumentos; invocar o método remoto correspondente ao serviço que está a ser substituído; receber resultados. Este teste vai permitir verificar se o componente ws está a funcionar correctamente."
