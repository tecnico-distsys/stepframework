These subprojects are used to test JAX-WS handlers (SOAP message interceptors) and related classes.


handlers-abcde is a library with several handlers (A..., B..., C..., D..., E...)

HelloWS is a Web Service
HelloWSClient is a console Web Service client
HelloWSWebClient is a web application Web Service client

docs contains test results and message context property samples.
